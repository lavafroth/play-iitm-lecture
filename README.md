## IITM: mpv plugin

> [!NOTE]
> Only tested on Linux

### Prerequisites
- [mpv](https://mpv.io)
- [Violentmonkey](https://violentmonkey.github.io/)

### Installation
- Add the `player.js` and `keybinding.js` scripts to Violentmonkey
- Run `setup.sh`

### Usage

- Visit a page with a lecture video
- Press dot to play it in mpv
